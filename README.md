# FamilyFlow - Home Assistant Add-on

Family task planner and reward system for Home Assistant.

## Installation

1. In Home Assistant, go to **Settings** → **Add-ons** → **Add-on Store**
2. Click **⋮** (three dots) in the top right → **Repositories**
3. Add this repository URL: `https://github.com/YOUR_USERNAME/familyflow-addon`
4. Click **Add** → **Close**
5. Find **FamilyFlow** in the add-on list
6. Click **Install**

## Configuration

After installation:
1. Click **Start** to run the add-on
2. Click **Open Web UI** to access the FamilyFlow interface

## About

This add-on uses pre-built Docker images from GitHub Container Registry. The images are automatically built from the private source repository via GitHub Actions.
